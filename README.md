# Responsive-image-gallery(INF-113)


A simple responsive image gallery built with HTML and CSS.
This project was created to practice frontend fundamentals such as page structure, image organization, responsive layout and visual styling.

## Preview

This project displays a gallery-style web page using static images and custom CSS.

## Features

* Static web page built with HTML and CSS
* Responsive image gallery layout
* Organized image folder
* Clean visual structure
* Basic frontend design practice
* Lightweight project without external frameworks

## Tech Stack

* HTML5
* CSS3

## How to Run

Clone the repository:

```bash
git clone https://github.com/ArianneRios/proyecto113ari.git
```

Open the project folder:

```bash
cd proyecto113ari
```

Open `index.html` directly in your browser.

## Project Purpose

The purpose of this project is to practice basic frontend development using HTML and CSS, focusing on visual layout, image presentation and responsive web design.
